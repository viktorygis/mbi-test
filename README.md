# MBI Test — диагностика и самопомощь при профессиональном выгорании

**MBI Test** — онлайн-инструмент для самодиагностики и отслеживания профессионального выгорания по методике Maslach Burnout Inventory (MBI). Методика используется для оценки эмоционального истощения, деперсонализации и редукции личных достижений [web:78][web:84].

## Возможности

- Диагностика по трем ключевым шкалам:
  - Психоэмоциональное истощение.
  - Деперсонализация.
  - Редукция личных достижений.
- Расчет общего индекса психического выгорания.
- Определение уровня риска по каждой шкале.
- Визуальный отчет с прогресс-барами, интерпретациями и рекомендациями.
- Экспорт результатов в PDF.

## Для кого

Приложение подходит специалистам помогающих профессий, менеджерам, HR-специалистам и всем, кто хочет отслеживать признаки эмоционального и профессионального истощения.

## Быстрый старт

```bash
git clone https://github.com/viktorygis/mbi-test.git
cd mbi-test
npm install
npm run dev
```

- Откройте приложение: **http://localhost:5173**
- Без backend результаты сохраняются только в браузере, в `localStorage`.

## Работа с данными теста

### `questions.json`

Можно безопасно редактировать вопросы, варианты ответов и подписи интерфейса.

### `scales.json`

Можно менять текстовые описания шкал, тексты рекомендаций и подсказки.
Осторожно изменяйте:

- список вопросов в шкалах (`items`);
- нормы (`norms`);
- лимиты (`maxScore`).

Эти поля влияют на расчеты и PDF-отчет. Не меняйте структуру ключей и вложенность, если не уверены в последствиях.
