# Тест MBI — Опросник на определение уровня психического выгорания

Веб-приложение для прохождения стандартизированного опросника **MBI (Maslach Burnout Inventory)** по К. Маслач и С. Джексон в адаптации Н.Е. Водопьяновой.

---

## Что делает приложение

Пользователь вводит свои данные (ФИО, телефон, e-mail), последовательно отвечает на 22 утверждения опросника, выбирая частоту каждого переживания по шкале от «Никогда» до «Каждый день». После завершения приложение рассчитывает:

- баллы по трём клиническим шкалам (психоэмоциональное истощение, деперсонализация, редукция личных достижений);
- **общий индекс психического выгорания** (0–132);
- нормативный уровень по каждому показателю (от «Крайне низкого» до «Крайне высокого»).

Результаты отображаются на отдельной странице с прогресс-барами, таблицами норм и содержательным описанием каждой шкалы.

## Методика

| Параметр | Значение |
|---|---|
| Авторы | К. Маслач, С. Джексон |
| Адаптация | Н.Е. Водопьянова |
| Число утверждений | 22 |
| Варианты ответов | Никогда (0) / Очень редко (1) / Иногда (3) / Часто (4) / Очень часто (5) / Каждый день (6) |

### Шкалы и ключ

| Шкала | Вопросы | Макс. балл | Интерпретация |
|---|---|---|---|
| Психоэмоциональное истощение | 1, 2, 3, 6, 8, 13, 14, 16, 20 | 54 | Чем выше — тем сильнее выгорание |
| Деперсонализация | 5, 10, 11, 15, 22 | 30 | Чем выше — тем сильнее выгорание |
| Редукция личных достижений | 4, 7, 9, 12, 17, 18, 19, 21 | 48 | **Обратная шкала**: чем ниже — тем сильнее выгорание |

### Формула общего индекса

```
Индекс выгорания = Истощение + Деперсонализация + (48 − Редукция)
Максимум: 54 + 30 + 48 = 132
```

### Нормативные диапазоны

**Истощение:**
| Крайне низкое | Низкое | Среднее | Высокое | Крайне высокое |
|---|---|---|---|---|
| 0–10 | 11–20 | 21–30 | 31–40 | 41–54 |

**Деперсонализация:**
| Крайне низкое | Низкое | Среднее | Высокое | Крайне высокое |
|---|---|---|---|---|
| 0–5 | 6–11 | 12–17 | 18–23 | 24–30 |

**Редукция личных достижений (обратная):**
| Крайне низкое выгорание | Низкое | Среднее | Высокое | Крайне высокое выгорание |
|---|---|---|---|---|
| 39–48 | 29–38 | 19–28 | 9–18 | 0–8 |

**Общий индекс выгорания:**
| Крайне низкое | Низкое | Среднее | Высокое | Крайне высокое |
|---|---|---|---|---|
| 0–23 | 24–49 | 50–75 | 76–101 | 102–132 |

## Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/viktorygis/mbi-test.git
cd mbi-test
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Запустить приложение

```bash
npm run dev
```

Приложение откроется по адресу **http://127.0.0.1:5173**

### 4. (Опционально) Запустить API-сервер

```bash
node server.js
```

Сервер запускается на порту **5000** и обеспечивает сохранение результатов теста (в памяти, для разработки).

> Без сервера результаты сохраняются в `localStorage` браузера.

## Поток данных

```
Пользователь → FormScreen → QuestionsScreen → PatternTestEntry
                                                    │
                                            sendResults({ user, answerIndices, date })
                                                    │
                                     localStorage  / API POST /api/test-mbi/submit
                                                    │
                                              /results/:id
                                                    │
                                      PatternTestResultPage
                                      fetch questions.json + localStorage
                                                    │
                                          createMbiResults(answerIndices, mbiData)
                                                    │
                                            ResultsScreen → MbiScalesSection
```

## Конфигурация данных опросника

Всё содержимое теста хранится в `public/data/questions.json`. Структура файла:

```json
{
  "questions": [ { "id": 1, "text": "..." }, ... ],
  "answerOptions": ["Никогда", "Очень редко", "Иногда", "Часто", "Очень часто", "Каждый день"],
  "scores": [0, 1, 3, 4, 5, 6],
  "scales": {
    "exhaustion":       { "items": [...], "maxScore": 54, "norms": [...] },
    "depersonalization":{ "items": [...], "maxScore": 30, "norms": [...] },
    "reduction":        { "items": [...], "maxScore": 48, "invertedForIndex": true, "norms": [...] }
  },
  "burnoutIndex": {
    "maxScore": 132,
    "norms": [...]
  }
}
```
