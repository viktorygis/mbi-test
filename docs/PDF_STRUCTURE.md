# MBI — Структура PDF-отчёта

> **Назначение:**
> Описание структуры, блоков и оформления PDF-отчёта для теста MBI (Maslach Burnout Inventory) —
> теста на профессиональное выгорание.
> Документ предназначен для разработчиков, дизайнеров и интеграторов.

---

## Общая структура PDF

- Внизу каждой страницы нумерация:
  **Страница N из M | ai4g.ru**
- Отступы страницы: `[40, 50, 40, 50]`
- Акцентный цвет: **#ff008a** (линии, выделения)
- Основной цвет ссылок: **#007BFF**
- Вспомогательный серый: **#71717a**

## Блок 2 (продолжение) — Расшифровка результата — Страница 2

Три шкалы, **отсортированные по приоритету проблемности** (от высокого к низкому):

| Шкала | Иконка | Особенность |
| ----- | ------ | ----------- |
| Эмоциональное истощение | `emotional` | Стандартная шкала |
| Деперсонализация | `depersonalization` | Стандартная шкала |
| Редукция личных достижений | `reduction` | Обратная шкала — подсказка «Чем выше балл — тем сильнее снижение» |

Для каждой шкалы выводится:

| Элемент | Описание |
| ------- | -------- |
| Иконка + название + уровень | Три колонки: иконка (14×14), название (`scaleTitle`), уровень цветом (`scaleLabel`) |
| Баллы | `N баллов из M`, стиль `scalePercent` |
| Линейка | Сегментированная (если есть нормы) или простая с `0% / X% / 100%` |
| Подс��азка | Только для `reduction` — курсивная подсказка про обратную шкалу |
| Рекомендации | `short` + `details`, стили `recoTitle` / `recoText` |

**Блок «Когда стоит обратиться за поддержкой»** — светлый фон `#fafafa`, текст с рекомендацией обратиться к специалисту.

**Разрыв страницы**

## Цвета уровней (`SEG_COLORS`)

Используются в сегментированных линейках:

| Ключ | Цвет | Описание |
| ---- | ---- | -------- |
| `veryLow` | `#4caf50` | Очень низкий — зелёный |
| `low` | `#8bc34a` | Низкий — жёлто-зелёный |
| `mid` | `#ffc107` | Средний — жёлтый |
| `high` | `#ff9800` | Высокий — оранжевый |
| `veryHigh` | `#f44336` | Очень высокий — красный |

```
[Шапка] → [Страница 1: Итоговый результат + Профиль]
        → [Страница 2: Расшифровка по шкалам + Поддержка]
        → [Страница 3: Справочная информация]
        → [Страница 4: Контакты + Источники]
```
