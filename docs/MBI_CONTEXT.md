# MBI Test — контекст для ИИ

## Назначение
MBI Test — опросник для оценки профессионального выгорания по трем шкалам:
1. Психоэмоциональное истощение.
2. Деперсонализация.
3. Редукция личных достижений.

Тест носит информационный характер и не является диагнозом.

## Входные данные
Пользователь отвечает на 22 вопроса.
Каждый ответ — число от 0 до 6.

## Перевод ответов в баллы
- никогда = 0
- очень редко = 1
- иногда = 3
- часто = 4
- очень часто = 5
- каждый день = 6

## Шкалы и ключи

### 1) Психоэмоциональное истощение
Вопросы: 1, 2, 3, 6, 8, 13, 14, 16, 20
Максимум: 54
Интерпретация: чем выше балл, тем сильнее истощение.

### 2) Деперсонализация
Вопросы: 5, 10, 11, 15, 22
Максимум: 30
Интерпретация: чем выше балл, тем сильнее отчуждение и циничность.

### 3) Редукция личных достижений
Вопросы: 4, 7, 9, 12, 17, 18, 19, 21
Максимум: 48
Это обратная шкала: чем ниже балл, тем выше выраженность выгорания и хуже чувство компетентности.

## Общий индекс выгорания
Формула:
Индекс = Истощение + Деперсонализация + (48 - Редукция)

Максимум индекса: 132.

## Нормы

### Истощение
- 0–10 — крайне низкое
- 11–20 — низкое
- 21–30 — среднее
- 31–40 — высокое
- 41–54 — крайне высокое

### Деперсонализация
- 0–5 — крайне низкое
- 6–11 — низкое
- 12–17 — среднее
- 18–23 — высокое
- 24–30 — крайне высокое

### Редукция личных достижений
- 39–48 — крайне низкая редукция, хорошее чувство эффективности
- 29–38 — низкая редукция
- 19–28 — средняя редукция
- 9–18 — высокая редукция
- 0–8 — крайне высокая редукция, выраженное выгорание

### Общий индекс
- 0–23 — крайне низкий
- 24–49 — низкий
- 50–75 — средний
- 76–101 — высокий
- 102–132 — крайне высокий

## Логика интерпретации
- Высокое истощение указывает на дефицит ресурса.
- Высокая деперсонализация указывает на эмоциональную дистанцию.
- Высокая редукция личных достижений указывает на снижение профессиональной самооценки.
- Комбинация высоких значений по нескольким шкалам усиливает риск выраженного выгорания.

## Правила ответа ИИ
1. Сначала посчитать баллы по каждой шкале.
2. Затем вычислить общий индекс.
3. Затем определить уровень по каждой шкале.
4. Затем дать короткую интерпретацию и рекомендации.
5. Если риски выражены слабо, дать профилактический вывод.
6. Если редукция низкая по баллу, обязательно подчеркнуть, что это означает более выраженное выгорание.
7. Не ставить диагноз и не заменять специалиста.
8. Отвечать кратко, структурно и по делу.

## Формат ответа
- Сначала: краткий итог.
- Затем: баллы по шкалам.
- Затем: общий индекс и его уровень.
- Затем: интерпретация.
- Затем: рекомендации.
