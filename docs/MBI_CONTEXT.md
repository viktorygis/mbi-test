# MBI Test — контекст для ИИ

## Назначение

**MBI Test (Maslach Burnout Inventory)** — это опросник для оценки признаков профессионального выгорания. Он помогает определить выраженность трех основных компонентов выгорания: эмоционального истощения, деперсонализации и редукции профессиональных достижений. Тест носит информационный характер и не заменяет консультацию специалиста.

Он помогает определить выраженность трех основных компонентов выгорания:

1. Эмоциональное истощение (_exhaustion_)
2. Деперсонализация (_depersonalization_)
3. Редукция личных достижений (_reduction_)

> ⚠️ Тест не заменяет консультацию специалиста и носит информационный характер.

---

## Предварительные вопросы

Перед основным тестом задаются дополнительные вопросы:

- возраст (`age`)
- род занятий (`occupation`)
- текущий приоритет (`priority`)

Эти ответы:

- не участвуют в расчётах,
- сохраняются отдельно,
- могут использоваться для персонализации рекомендаций.

---

## Входные данные (основной тест)

- 22 вопроса
- каждый ответ сохраняется как индекс (0–5)

### Соответствие ответов и индексов

| Индекс | Ответ       | Балл |
| ------ | ----------- | ---- |
| 0      | никогда     | 0    |
| 1      | очень редко | 1    |
| 2      | иногда      | 3    |
| 3      | часто       | 4    |
| 4      | очень часто | 5    |
| 5      | каждый день | 6    |

---

## ScoreMap (используется в коде)

```js
[0, 1, 3, 4, 5, 6];
```

---

## Шкалы и расчёт

### 1. Эмоциональное истощение (_exhaustion_)

- Вопросы: 1, 2, 3, 6, 8, 13, 14, 16, 20
- Формула: сумма баллов без изменений (ScoreMap)
- **Максимум:** 54

### 2. Деперсонализация (_depersonalization_)

- Вопросы: 5, 10, 11, 15, 22
- Формула: сумма баллов без изменений (ScoreMap)
- **Максимум:** 30

### 3. Редукция личных достижений (_reduction_)

- Вопросы: 4, 7, 9, 12, 17, 18, 19, 21
- Баллы по этим вопросам обязательно **инвертируются**:
  ```
  reduction_score = 6 - baseScore
  ```
  где:
  - `baseScore` — балл по ScoreMap для ответа [0,1,3,4,5,6]
  - итоговые значения для каждого ответа будут [6,5,4,3,1,0]
- Суммировать инвертированные баллы по этим вопросам (итог: 0–48).

## Итоговый индекс выгорания

- Формула:
  ```
  burnoutIndex = exhaustion + depersonalization + reduction
  ```
- **Максимум:** 132

## Интерпретация шкал

### Эмоциональное истощение (exhaustion)

- 0–10 — крайне низкое
- 11–20 — низкое
- 21–30 — среднее
- 31–40 — высокое
- 41–54 — крайне высокое

### Деперсонализация (depersonalization)

- 0–5 — крайне низкое
- 6–11 — низкое
- 12–17 — среднее
- 18–23 — высокое
- 24–30 — крайне высокое

### Редукция личных достижений (reduction) — уже ИНВЕРТИРОВАНО!

**Чем меньше баллов — тем лучше; чем больше — тем хуже!**

| Баллы | Текст для пользователя                                            |
| ----- | ----------------------------------------------------------------- |
| 0–8   | Крайне низкая редукция (норма, самооценка сохранена, это хорошо!) |
| 9–18  | Низкая редукция                                                   |
| 19–28 | Средняя редукция                                                  |
| 29–38 | Высокая редукция (зона риска)                                     |
| 39–48 | Крайне высокая редукция (очень плохо, выраженное выгорание)       |

#### Примеры формулировок:

- 7/48 — Крайне низкая редукция (всё хорошо, самооценка сохранена)
- 42/48 — Крайне высокая редукция (зона риска, проявилось сильное выгорание)

### Общий индекс выгорания

Сумма по трём шкалам (уже после инверсии баллов для редукции):

| Баллы   | Интерпретация         |
| ------- | --------------------- |
| 0–23    | Крайне низкий индекс  |
| 24–49   | Низкий индекс         |
| 50–75   | Средний индекс        |
| 76–101  | Высокий индекс        |

## Алгоритм расчёта

1. Получить `answerIndices[22]`

2. Для каждой шкалы посчитать сумму:
   - exhaustion: по выбранным вопросам, без инверсии.
   - depersonalization: по выбранным вопросам, без инверсии.
   - reduction: по выбранным вопросам, **с инверсией баллов** как `6 - baseScore` (см. выше).

3. Рассчитать общий индекс:

   ```
   burnoutIndex = exhaustion + depersonalization + reduction
   ```

   где для редукции используются уже инвертированные баллы.

4. По результату каждого блока и по сумме определить интервал и текстовое значение согласно таблицам выше.

5. Сформировать итог для пользователя:
   - краткий вывод
   - баллы по шкалам
   - индекс
   - интерпретация (каждая шкала + индекс)
   - рекомендации

---

## Ключевые правила интерпретации

- Высокое истощение → дефицит ресурсов
- Высокая деперсонализация → эмоциональное отстранение
- Для редукции: НИЗКИЕ баллы = хорошая самооценка, ВЫСОКИЕ баллы = выраженное выгорание!
- МНОГО баллов по шкале редукции — это ПЛОХО.

---

## Критические комбинации ("тревожные" формулы)

| Комбинация                     | Значение                     |
| ------------------------------ | ---------------------------- |
| истощение + деперсонализация ↑ | активная фаза выгорания      |
| истощение + высокая редукция   | потеря мотивации             |
| все шкалы высокие              | выраженный синдром выгорания |

---

## Важно

> ⚠️ Для шкалы редукции личных достижений: много баллов — это плохо!
> Повторная инверсия не допускается.
> Все выводимые формулировки должны передавать смысл: низкая редукция = хорошо, высокая редукция = тревожный признак.
