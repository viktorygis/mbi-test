# MBI Test — контекст для ИИ

## Назначение

**MBI Test (Maslach Burnout Inventory)** — онлайн-опросник для самодиагностики уровня профессионального выгорания.

Оцениваются три ключевых компонента:

1. Эмоциональное истощение (*exhaustion*)
2. Деперсонализация (*depersonalization*)
3. Редукция профессиональных достижений (*reduction*)

> ⚠️ Тест носит информационный характер, не является медицинской диагностикой и не заменяет консультацию специалиста.

---

## Предварительные вопросы

Перед основным тестом задаются дополнительные вопросы:

- возраст (`age`)
- род занятий (`occupation`)
- текущий приоритет (`priority`)

**Они:**
- не участвуют в расчётах,
- сохраняются отдельно,
- могут использоваться для персонализации рекомендаций.

## Входные данные (основной тест)

- 22 вопроса
- каждый ответ сохраняется как **индекс (0–5)**

### Соответствие ответов и индексов

| Индекс | Ответ         | Балл |
|--------|--------------|------|
| 0      | никогда      | 0    |
| 1      | очень редко  | 1    |
| 2      | иногда       | 3    |
| 3      | часто        | 4    |
| 4      | очень часто  | 5    |
| 5      | каждый день  | 6    |

## ScoreMap (используется в коде)

```js
[0, 1, 3, 4, 5, 6]
```

## Шкалы и расчёт

### 1. Эмоциональное истощение (*exhaustion*)
- Вопросы: 1, 2, 3, 6, 8, 13, 14, 16, 20
- Формула: сумма баллов без изменений
- **Максимум:** 54

### 2. Деперсонализация (*depersonalization*)
- Вопросы: 5, 10, 11, 15, 22
- Формула: сумма баллов без изменений
- **Максимум:** 30

### 3. Редукция профессиональных достижений (*reduction*)
- Вопросы: 4, 7, 9, 12, 17, 18, 19, 21
- **Важно:**
  Баллы инвертируются:
  ```
  score = 6 - baseScore
  ```
  где:
  - `baseScore` — значение из scoreMap
  - `6` — максимум шкалы

## Итоговый индекс выгорания

- Формула:
  ```
  burnoutIndex = exhaustion + depersonalization + reduction
  ```
- **Максимум:** 132

## Интерпретация шкал

### Эмоциональное истощение
- 0–10 — крайне низкое
- 11–20 — низкое
- 21–30 — среднее
- 31–40 — высокое
- 41–54 — крайне высокое

### Деперсонализация
- 0–5 — крайне низкое
- 6–11 — низкое
- 12–17 — среднее
- 18–23 — высокое
- 24–30 — крайне высокое

### Редукция (уже инвертированная!)
- 39–48 — низкое выгорание (норма)
- 29–38 — умеренное
- 19–28 — среднее
- 9–18 — высокое
- 0–8 — крайне высокое выгорание

### Общий индекс
- 0–23 — крайне низкий
- 24–49 — низкий
- 50–75 — средний
- 76–101 — высокий
- 102–132 — крайне высокий

## Алгоритм расчёта

**Шаг 1:**
Получить `answerIndices[22]`

**Шаг 2:**
Рассчитать шкалы:
```js
exhaustion = computeScaleScore(..., invert=false)
depersonalization = computeScaleScore(..., invert=false)
reduction = computeScaleScore(..., invert=true)
```

**Шаг 3:**
Рассчитать индекс:
```js
burnoutIndex = exhaustion + depersonalization + reduction
```

**Шаг 4:**
Определить уровни по таблицам интерпретации

**Шаг 5:**
Сформировать результат:
- краткий вывод
- баллы по шкалам
- индекс
- интерпретация
- рекомендации

---

## Ключевые правила интерпретации

- высокое истощение → дефицит ресурсов
- высокая деперсонализация → эмоциональное отстранение
- НИЗКАЯ редукция = ВЫСОКОЕ выгорание (обратная шкала!)

---

## Критические комбинации (клинические "тревожные" формулы)

| Комбинация                     | Значение                   |
|--------------------------------|----------------------------|
| истощение + деперсонализация ↑ | активная фаза выгорания    |
| истощение + низкая редукция    | потеря мотивации           |
| все шкалы высокие              | выраженный синдром выгорания|
