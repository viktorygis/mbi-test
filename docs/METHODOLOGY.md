# Методика теста MBI

## Что делает приложение

Пользователь вводит свои данные (ФИО, телефон, e-mail), последовательно отвечает на 22 утверждения опросника, выбирая частоту проявления состояния. После прохождения опроса приложение рассчитывает:

- баллы по трём клиническим шкалам (психоэмоциональное истощение, деперсонализация, редукция личных достижений);
- **общий индекс психического выгорания** (0–132);
- нормативный уровень по каждому показателю (от «Крайне низкого» до «Крайне высокого»).

Результаты отображаются на отдельной странице (см. компоненты `MbiResultPage`, `ResultsScreen`, `MbiScalesSection`), визуализируются прогресс-барами и снабжаются интерпретацией с подробными рекомендациями.

## Шкалы и ключ

| Шкала                        | Вопросы                       | Макс. балл | Интерпретация                                        |
| ---------------------------- | ----------------------------- | ---------- | ---------------------------------------------------- |
| Психоэмоциональное истощение | 1, 2, 3, 6, 8, 13, 14, 16, 20 | 54         | Чем выше — тем сильнее выгорание                     |
| Деперсонализация             | 5, 10, 11, 15, 22             | 30         | Чем выше — тем сильнее выгорание                     |
| Редукция личных достижений   | 4, 7, 9, 12, 17, 18, 19, 21   | 48         | **Обратная шкала**: чем ниже — тем сильнее выгорание |

---

## Формула общего индекса

```
Индекс выгорания = Истощение + Деперсонализация + (48 − Редукция)
Максимум: 54 + 30 + 48 = 132
```

## Как происходит интерпретация результатов (алгоритм)

1. **Подсчёт баллов** по каждой шкале (см. ключи выше).
2. **Рассчитывается общий индекс** по формуле:
   `Общий_индекс = Истощение + Деперсонализация + (48 – Редукция)`
3. **Для каждой шкалы** определяется диапазон результата (норма) по таблицам выше.
4. **Формируется индивидуальная интерпретация**:
   - Если уровень "истощения" высокий — рекомендация делать паузы, работать с восстановлением, снижать нагрузку.
   - Если высокая "деперсонализация" — рекомендации по поддержке эмпатии, снижению дистанции.
   - Если "редукция достижений" низкая — рекомендации по повышению самоэффективности и фиксации успехов.
5. **Дополнительно**: показывается подробная справочная информация о шкалах и советы по профилактике выгорания.
6. **Всё это реализует компонент**
   - `src/components/Sections/ResultsSections/MbiScalesSection.jsx` (визуализация шкал, прогресс-бары)
   - `src/components/Sections/ResultsSections/MbiInterpretationSection.jsx` (интерпретация)
   - `src/components/Sections/ResultsSections/MbiRecommendationsSection.jsx` (рекомендации)
   - Для генерации PDF: `src/utils/pdf/mbiPdfGenerator.js`
